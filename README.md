# Projeto Cordel 📖

Bem-vindo ao repositório do **Projeto Cordel**! Este projeto é uma homenagem e reprodução digital inspirada no cordel de Milton Duarte, construído com foco em estruturação semântica e estilização visual.

## 📝 Sobre o Projeto

O objetivo principal deste projeto é trazer a literatura de cordel de Milton Duarte para o ambiente web, aplicando boas práticas de desenvolvimento front-end com **HTML5** e **CSS3**. O layout foi pensado para oferecer uma leitura agradável e imersiva.

## 🛠️ Tecnologias Utilizadas

As seguintes tecnologias foram utilizadas na construção deste projeto:

* **HTML5** - Para a semântica e estruturação do texto do cordel.
* **CSS3** - Para a estilização, tipografia e apresentação visual.
